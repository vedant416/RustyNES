RustyNES
RustyNES is a NES emulator written in Rust.
It is written from scratch without using any external libraries.
This is a web interface for the emulator running in the browser using WebAssembly.
Click on the toggle mute button to enable sound.
For best experience set refresh rate to 60Hz.
Otherwise sound will not be clear.
View the source code on GitHub: Source Code
Website is currently under development.
